# 🚀 Consumo de APIs RESTful - Laboratorio 2 y 3

Aplicación web interactiva desarrollada para el Laboratorio 2 y 3 de la asignatura Desarrollo de Aplicaciones Web. El proyecto demuestra la integración de APIs externas (PokeAPI y JSONPlaceholder) mediante peticiones asíncronas con JavaScript (Fetch API) y un diseño moderno utilizando Bootstrap 5.

## 🛠️ Tecnologías utilizadas
* **HTML5 & CSS3** - Estructura y diseño base.
* **Bootstrap 5** - Framework para el diseño responsivo.
* **JavaScript (Fetch API)** - Lógica para el consumo de APIs y manipulación del DOM.
* **Netlify** - Hosting para el despliegue.

## 📋 Funcionalidades
- **Batalla Pokémon:** Consumo de PokeAPI para mostrar estadísticas y sprites.
- **Lista de Usuarios:** Consumo de JSONPlaceholder para listar perfiles.
- **Diseño Responsivo:** Adaptable a dispositivos móviles.

- [x] Configuración del entorno de desarrollo.
- [x] Maquetación de la interfaz con Bootstrap.
- [x] Conexión y extracción de datos de PokeAPI.
- [x] Renderizado de lista de usuarios desde JSONPlaceholder.
- [x] Despliegue exitoso en Netlify.
